# STYLE.md — 排版与写作风格指南

## 语言与编码

- 全文使用 **UTF-8** 编码，无 BOM。
- 使用中文标点（，。：；？！""『』）全角标点。
- 英文与数字使用半角字符，前后加空格。
- 避免中英混杂的句子，名词统一。

## 标题体系

```markdown
# 第一章 肝经          ← 章节级（章名独立一行）
## 第一节 太冲          ← 穴位级
### 别名               ← 子字段
#### （不使用四级以下标题）
```

- `#` 与标题文字之间留一个空格。
- 禁止跳级（如 `#` 之后直接到 `###`）。

## 模板字段顺序

严格按以下顺序使用，不得删减或重排：


## 正文写作风格

- 使用第三人称客观叙述，避免"我认为""我们觉得"。
- 医学内容（正文、译文、总结）由 GPT 编写，工程师不得修改医学观点。
- 仅负责格式、排版、错别字修正。
- 条目内容为空白时，保留标题并写明：`（内容待作者提供）`

## 古籍原文格式

```markdown
> **《灵枢·九针十二原》**
> 阴中之少阳，肝也，其原出于太冲。
```

- 原文用 `>` 引用块包裹。
- 书名加粗。
- 不擅自翻译原文，保持原文独立呈现。

## 古籍译文格式

- 译文放在原文之后，标题为 `## 古籍译文`。
- 译文中关键术语第一次出现时保留原文：
  > 肝气郁结（肝气郁结）
- 译文风格：直译为主，忠实原文，不加文采润色。

## 脚注与参考文献

- 脚注编号用 `[^1]`，`[^2]` 顺序排列。
- 文末参考文献按出现顺序编号。
- 格式示例：
  ```markdown
  [^1]: 《灵枢·九针十二原》，人民卫生出版社，2012，第 23 页。
  ```

## 标点与空格

| 场景 | 规范 |
|------|------|
| 中文文字后 | 用中文标点，不加空格 |
| 数字 + 单位 | 数字与单位之间不加空格（如 3cm） |
| 括号 | 全角括号（） |
| 引号 | 先双引号""，内嵌单引号'' |
| 省略号 | 使用中文省略号「……」而非「...」 |

## 专有名词

- 经络名称：足厥阴肝经、手少阴心经（全称）
- 穴位名称：太冲、神门（首次全称，重复时可用简称）
- 书名号：《》《》
- 避免口语化表达（如"咱们"、"大家"）

## 换行

- 段内不手动换行，由渲染引擎自动排版。
- 段落之间空一行。
- 列表项之间空一行。
- 引用块前后各空一行。

## 图片

- 图片存于 `images/` 目录。
- 引用路径使用相对路径：`../images/xxx.png`
- 图注格式：`*图1：太冲穴定位示意图*`
- 优先使用矢量图（SVG），其次 PNG。
