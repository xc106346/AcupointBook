# WritingRules.md — 正文写作规则

## 编写责任边界

- **Codex 禁止**：编写医学内容、修改 GPT 正文的医学观点、编造古籍原文。
- **GPT 负责**：提供所有穴位正文、古籍译文、医理总结。
- **作者负责**：医案、临床经验、图片资料。
- **GPT + Codex 协作**：统一全稿格式、修正错别字、统一术语。

## 模板使用规范

穴位正文严格使用 `chapters/_TEMPLATE.md` 中的 22 字段模板。

不得新增字段，不得删除字段，不得改变字段顺序。

模板字段：
```
01 穴名           ← 穴位名称（中文）
02 拼音           ← 拼音全拼
03 别名           ← 别称（如有）
04 出处           ← 首次记载的典籍
05 所属经络       ← 经脉全称
06 穴性           ← 腧穴性质
07 五行属性       ← 五行归属
08 定位           ← 解剖定位
09 取穴方法       ← 实际操作取穴步骤
10 针刺           ← 针刺深度、角度、方向
11 艾灸           ← 艾灸方法与时间
12 功效           ← 主要功效
13 主治           ← 主治病症
14 古籍原文       ← 引用原文，不改写
15 古籍译文       ← 直译，不润色
16 古籍应用总结   ← GPT 综合历代观点
17 经典配穴       ← 经典配穴方案
18 现代临床应用   ← 现代研究与临床
19 作者按         ← 作者个人经验（仅作者填写）
20 注意事项       ← 禁忌与注意事项
21 记忆口诀       ← 临床记忆方法
22 参考文献       ← 引用文献列表
```

## 古籍原文引用规则

1. 仅引用本数据库（FullTextIndex.sqlite）中存在的段落。
2. 每条引用注明：书名、篇名、段落内容。
3. 引用时保留原文全貌，不做删节（除非有明显笔误需注释说明）。
4. 同一典籍多次引用，在首次引用时详细标注，后续引用可简写。

## 译文写作规则

1. **直译优先**：忠实原文结构，不添加原文没有的信息。
2. **术语保留**：关键中医术语首次出现时附原文。
3. **不删不增**：不删减原文内容，不添加译者的个人理解。
4. **分节呈现**：长段原文按意义分节翻译，每节对应一段原文。

## 现代内容编写规则

- 功效、主治等栏目使用结构化列表：
  ```markdown
  - 头痛（肝阳上亢所致）
  - 眩晕（肝风内动）
  ```
- 避免在正文中出现"近年来""目前""一般来说"等模糊时间词。
- 配穴部分：先列穴名，再简述配伍原理。

## 质量控制

- 提交前朗读一遍，检查语义是否通顺。
- 检查字段是否有空内容（空内容写"（内容待提供）"）。
- 检查书名号、引号是否成对。
- 检查参考文献格式一致性。
