## 23 字段结构

1. 穴名
2. 拼音（含国际代号）
3. 别名（含表格）
4. 出处
5. 所属经络
6. 穴性（五输穴类别）
7. 五行属性
8. 定位（标准 + 解剖标志 + 古籍原文 + 辨析）
9. 取穴方法（体位 + 步骤 + 歌诀）
10. 针刺（操作表 + 注意事项）
11. 艾灸（方法表 + 适应证 + 禁忌）
12. 功效（核心 + 传统归纳 + 病机对应）
13. 主治（按系统分类）
14. 古籍原文（按年代排序：灵枢→难经→甲乙经→千金→铜人→资生经→大成→医宗金鉴→近现代教材，每条标出处）
15. 古籍译文（按同序逐条白话翻译）
16. 历代医家（按时代分类：唐宋/金元/明/清/近现代/现代）
17. 古籍应用总结（综合学术分析）
18. 经典配穴（按疾病分类：疝气/月经病/遗尿/癫狂/昏迷/肝胆病等，每组注明出处）
19. 现代临床应用（循证医学总结：现状/证据充分/证据不足）
20. 学术按语（不是心得，分析：为什么是井穴/为什么属木/古今变化/常与何穴配伍/放血机理）
21. 注意事项
22. 记忆口诀
23. 参考文献（统一编号 [1][2]...，去重，规范格式）

## 写作规范

### 古籍引用规范
- 必须按年代顺序：灵枢 → 难经 → 针灸甲乙经 → 千金要方 → 铜人腧穴针灸图经 → 针灸资生经 → 针灸大成 → 医宗金鉴 → 近现代教材 → 现代文献
- 每条引用前用引用块 `>` 标记
- 末尾注明出处，例：（《灵枢·本输》）

### 历代医家格式
```
### 唐宋时期
- **王惟一**（《铜人腧穴针灸图经》）：……

### 明代
- **杨继洲**（《针灸大成》）：……
```

### 配穴按疾病分类格式
```
#### 1. 疝气
| 配穴 | 出处 | 机理 |
|------|------|------|
| 大敦 + 太冲 | 《针灸大成》 | 井输配伍，疏肝理气 |
```

### 学术按语要点
- ① 五输穴性：为什么是井穴
- ② 五行属性：为什么属木
- ③ 古今应用演变
- ④ 经典配伍：常与何穴配伍及原理
- ⑤ 特色疗法：放血/温灸等机理

### 现代临床循证总结格式
```
#### 当前主要应用方向
- 方向 1：……
- 方向 2：……

#### 证据较充分
- 证据 1（来源）
- 证据 2（来源）

#### 证据不足或缺乏高质量 RCT
- 领域 1：……
```

### 参考文献格式
- 古代文献：[1] 《灵枢·本输》. 战国时期.
- 现代教材：[2] 《经络腧穴学》. 中国中医药出版社, 2012.
- 期刊文献：[3] 张三. 大敦穴临床应用研究. 中国针灸, 2020, 40(5): 12-15.

### Markdown 统一
- 一级标题：`# 大敦`
- 二级标题：`## 一、穴名`
- 三级标题：`### 1. 肝胆病症`
- 引用块：`> 原文`
- 表格：每列对齐
- 段落：空行隔开
- 列表：`-` 一致

## 重复内容处理

功效 / 主治 / 现代临床 / 学术按语 需明确分工：
- 功效：中医术语化的功能（如疏肝理气、清热利湿）
- 主治：现代医学病名 + 中医证候
- 现代临床：循证医学证据总结
- 学术按语：作者学术分析

不允许：四节中出现相同句子。
