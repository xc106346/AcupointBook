# CitationRules.md — 引用与参考文献规范

## 引用来源

仅允许引用以下来源：
- 本地全文数据库 `FullTextIndex.sqlite` 中已收录的文献
- 作者提供的原始资料（须在 `references/source-map.yaml` 中登记路径）
- 经作者确认的临床资料

禁止引用：
- 网络搜索结果（无来源保障的内容）
- 未经确认的 Wikipedia、百度百科等
- 模型自行生成的"古籍内容"（非真实引文）

## 脚注格式

### 格式

```markdown
阴中之少阳，肝也，其原出于太冲。[^1]

[^1]: 《灵枢·九针十二原》，人民卫生出版社影印本，2012，第 34 页。
```

### 脚注编号

- 从 `[^1]` 开始，不重复编号。
- 同一来源多次引用，重复使用同一编号。
- 脚注内容放在正文末尾、参考文献之前。

## 参考文献格式

### 格式规范

所有参考文献统一采用以下格式，**不得混用其他格式**：

```
[编号] 作者（时代）。《书名》[M]. 出版地：出版社，年份。
```

### 示例

```
[1] 佚名。《灵枢经》[M]. 北京：人民卫生出版社，2012.
[2] 王叔和（魏）。《脉经》[M]. 北京：人民卫生出版社，2011.
[3] 孙思邈（唐）。《备急千金要方》[M]. 北京：人民卫生出版社，2011.
[4] 杨继洲（明）。《针灸大成》[M]. 北京：中国中医药出版社，2008.
[5] 黄帝，岐伯等。《针灸甲乙经》[M]. 北京：人民卫生出版社，2011.
```

### 字段说明

| 字段 | 说明 |
|------|------|
| [M] | 专著标识 |
| [J] | 期刊文章 |
| [N] | 报纸文章 |
| [D] | 学位论文 |

### 作者格式

- 古代文献：作者（时代）
- 现代文献：作者1, 作者2, 作者3

## 引文质量检查

每次引用前核查：
1. 引文是否来自真实典籍（非模型生成）
2. 引文内容是否与原文一致（无误字、脱漏）
3. 书名、篇名、卷数是否正确
4. 页码是否可查

## 引用登记

所有新引用的文献须在 `references/source-map.yaml` 中登记：

```yaml
《灵枢经》:
  path: "G:\\中医文库\\书籍\\灵枢经"
  type: "ancient"
  dynasty: "战国"
  compiler: "佚名"
```

## 重复引文

同一穴位多次引用同一典籍时，第一次详细标注（含页码），后续简写为书名+篇名即可。

## 多文献综合引用

多来源综合叙述时，在段末统一列出参考文献：

```markdown
太冲为肝经原穴，主疏肝解郁[^1]、平肝熄风[^2]、理气止痛[^3]。

[^1]: 《灵枢·九针十二原》
[^2]: 《针灸甲乙经》
[^3]: 《针灸大成》
```
