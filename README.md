# 《五输穴与原穴经典汇编》项目工程说明

本目录只负责图书工程，不编写任何医学内容。

## 当前状态

- 已建立项目骨架、章节目录、参考文献目录和输出目录。
- 已建立统一 Markdown 模板。
- 已为目前明确列出的穴位建立空白稿。
- 其余章节内容等待作者提供后补入。

## 约定

- 所有穴位稿件均使用统一模板。
- 正文、译文、应用总结等栏目在作者提供内容前保持空白。
- 引文统一采用脚注格式，文末统一列出参考文献。
- EPUB、DOCX、PDF 由后续生成流程产出，输出目录固定为 `output/`。
- 总文献库 `G:\中医文库\书籍\` 默认不扫描、不建索引。
- 只有用户明确指定书名或完整路径时，才读取对应文献。
- 已读取文献只在 `references/source-map.yaml` 记录路径映射，不保留正文副本。

## 目录用途

- `references/`：存放各典籍来源材料。
- `images/`：存放插图、封面和版式辅助资源。
- `chapters/`：存放各章节与穴位稿件。
- `output/`：存放最终导出文件。
- `docs/`：存放写作、引用和工作流规则。
