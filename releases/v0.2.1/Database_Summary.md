## 使用说明

**全文检索语法（FTS5）：**
```sql
SELECT * FROM fts WHERE text MATCH '关键词'
```

**典型检索词结构（以「太冲」为例）：**
```
太冲                         → 穴名本身
足厥阴肝经                   → 所属经络
输穴 OR 原穴                 → 五输穴类别
疏肝 OR 平肝 OR 理气         → 功效关键词
头痛 OR 眩晕 OR 高血压        → 主治关键词
太冲 合谷 OR 太冲 三阴交     → 配穴检索
```
